## Install

```sh
npm install @thoo0224/tebex-headless
yarn add @thoo0224/tebex-headless # for yarn
pnpm add @thoo0224/tebex-headless # for pnpm
```

## Run tests

```sh
npm run test
yarn test # for yarn
pnpm test # for pnpm
```
